Welcome to Headlines documentation

POST a headline:

/headline

Sample body

{
    "article": null,
    "userId": "546a9404e3650bc830d62466",
    "headline": "RHCP are the bomb",
    "dateCreated": 9765467,
    "voteCount": 0,
    "threshold": 5
}

GET the 20 latest headlines

/headlines/date/:number

Examples

/headlines/date/0

will get the latest 20 headlines

/headlines/date/10

will get the 10th - 30th latest headlines

Example Response

[
   {
       "_id": "54b0d293036afeca6a640965",
